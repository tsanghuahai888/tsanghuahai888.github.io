免费博客搭建

2019-07-13
服务

利用 GitHub 搭建 blog，减少服务器维护成本。

github环境步骤：

1.创建username.github.io仓库，地址就是这个。username是GitHub的username

2.随便添加个index.html。URL地址是上面的名字，如果打不开可以在settings找到github pages。点一下就能打开。

3.用自己的域名解析，需要设置Custom domain。

gitee环境步骤：

安装NodeJs需要配置插件仓库。

因为NodeJs使用国外仓库，下载插件相当慢。

使用node命令安装hexo框架

样式

去github上找，将样式clone 到 \blog\themes，修改_cofing.yml的themes

维护

blog文件目录：/source/_posts

删除文章
